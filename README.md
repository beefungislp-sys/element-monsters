# ⚗️ Element Monsters 🐉

Juego de cartas educativo de química, tipo Yu-Gi-Oh, para niños. Cada elemento de la
tabla periódica es un monstruo. ¡Combínalos para formar compuestos reales y gana batallas!

## 🎮 Jugar ahora

👉 **[Jugar en el navegador](https://beefungislp-sys.github.io/element-monsters/)**

No necesitas instalar nada. Funciona en compu, tablet o celular.

## Modos

- 🔬 **Laboratorio** — Explora los 36 monstruos (Hidrógeno → Kriptón), aprende qué es
  cada elemento con un dato curioso, y descubre con quién se combina.
- ⚔️ **Batalla vs Compu** — Por turnos, tipo Yu-Gi-Oh. Forma **monstruos** (atacan),
  **escudos** (bloquean) y **hechizos** (efectos). ¡Deja al rival en 0 de vida!

## Qué se aprende

- Símbolos y nombres de los elementos.
- Qué es cada uno (gas, metal, etc.) y su familia en la tabla periódica.
- Fórmulas de compuestos reales (H₂O, NaCl, CO₂, Fe₂O₃...).

## Cómo está hecho

Un solo archivo `index.html` (HTML + CSS + JavaScript, sin librerías). Los monstruos
son dibujos hechos con SVG y los sonidos se generan con Web Audio — sin archivos pesados.
